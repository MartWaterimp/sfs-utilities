About

Placidity's SFS Utilities is an unofficial collection of browser tools for Spaceflight Simulator. The goal is simple: useful calculators, human wording, and a clean Material Design 3 look.

palette

Material Design 3

The site uses Google's Material Design 3 language: tonal surfaces, rounded shapes, filled and tonal buttons, and a calm color system that stays readable on small screens.

cookie

Cookie-based saves

Custom fuels from the calculator are stored in browser cookies so your ratios and dry mass fractions come back the next time you open the page.

view_module

Modular layout

Shared pieces like the navbar and footer live in separate files under modules/, so every page can pull in the same chrome without copy-pasting markup.

code

Open and expandable

Pages are plain HTML, CSS, and JavaScript. That makes it easy to host on GitHub Pages and add more utilities later.
